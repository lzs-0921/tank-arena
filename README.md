# Tank Arena

经典坦克大战游戏 - 使用 HTML + CSS + 纯 JavaScript 实现。

## 操作说明

- **移动**: WASD 或 方向键
- **射击**: 空格键

## 功能特性

- 玩家坦克 vs AI 敌方坦克
- 多种地形：砖墙、钢墙、草地、水体
- 基地保护模式
- 多关卡递进
- 计分与生命系统

## 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏。

## 技术栈

- HTML5 Canvas
- CSS3
- Vanilla JavaScript (ES6+)

## 开发

本项目为纯前端实现，无需构建工具。建议使用 VS Code + Live Server 插件进行开发。
