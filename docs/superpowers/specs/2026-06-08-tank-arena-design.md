# Tank Arena - Cyberpunk Roguelike 设计文档

## 概述

经典坦克大战的现代重制：赛博朋克科幻 + 房间制 Roguelike + 双摇杆操控。纯前端 HTML + CSS + JS (ES6 Modules) 实现，零构建依赖。

---

## 核心循环

```
MAIN_MENU → RUNNING → ROOM_CLEAR → SHOP → RUNNING (下一房间) → BOSS → DEATH → RESULT → MAIN_MENU
```

---

## 架构：ES6 Modules

### 文件结构

```
tank-arena/
├── index.html
├── style.css
└── js/
    ├── main.js          # 入口：初始化、游戏主循环、状态机
    ├── config.js         # 全局常量
    ├── input.js          # 键盘+鼠标输入抽象
    ├── entities.js       # 坦克/子弹/敌人/拾取物 实体类
    ├── terrain.js        # 交互地形：掩体/屏障/传送带/爆炸桶
    ├── room.js           # 房间生成器：随机布局+门+敌人配置
    ├── combat.js         # 碰撞检测、伤害计算、弹道
    ├── shop.js           # 商店逻辑：碎片经济、升级池、刷新
    ├── meta.js           # 局外成长：数据核心、天赋树、解锁管理
    ├── renderer.js       # Canvas 渲染：霓虹光效、粒子系统
    ├── hud.js            # 游戏内 HUD
    └── overlay.js        # 主菜单/商店界面/死亡结算
```

**原则**：模块间通过 GameState 对象通信。实体/地形/房间是纯逻辑，渲染只读状态。

---

## 操控

双摇杆模式：WASD 移动车体 + 鼠标瞄准控制炮塔旋转。炮塔有转速限制，不可瞬间掉头。玩家碰撞体为圆形。

## 战斗系统

### 子弹类型
| 类型 | 行为 |
|------|------|
| 普通弹 | 直线飞行，碰墙/敌人消失 |
| 反弹弹 | 碰到墙壁反弹 2 次 |
| 穿透弹 | 穿透敌人继续飞行 |
| 散射弹 | 一次发射 3-7 颗扇形扩散 |
| 爆破弹 | 命中产生小范围爆炸 |
| 能量弹 | 无视可破坏掩体，被能量屏障反弹 |

### 敌人
| 名称 | 行为 |
|------|------|
| 巡逻无人机 | 移动慢，直线射击，杂兵 |
| 突击机甲 | 追踪玩家，二连射 |
| 狙击炮台 | 固定，远程高伤低速弹 |
| 自爆蜘蛛 | 高速冲撞自爆 |
| 护盾守卫 | 正面持能量盾，需要绕后 |
| 骇客节点 | 不攻击，给房间内敌人加攻速 buff |
| Boss | 每层关底，大型机甲，多阶段 |

### 碰撞
AABB 粗筛 → 圆形细节碰撞。实体间推开避免重叠。无友伤。

## 房间系统

### 类型
| 类型 | 权重 | 说明 |
|------|------|------|
| 普通战斗 | 50% | 标准敌人，清完开门 |
| 精英战斗 | 15% | 更强组合，2x 碎片 |
| 商店 | 每层固定 1 个 | 碎片兑换升级 |
| 宝藏 | 8% | 随机免费升级或大量碎片 |
| 事件 | 7% | 风险/收益文本抉择 |
| Boss | 每层末尾 | 大型 Boss |

### 布局
- 游戏画布固定 960×640，每个房间铺满整个画布，边界为钢墙
- 房间内随机 3-8 个交互地形元素
- 2-3 个门，清完敌人后开启并高亮
- 1-3 波敌人生成

### 地图
每层 8-12 个房间随机连接图，玩家可选路径。

### 主题层
| 层 | 主题 | 主色调 | 地形特征 |
|----|------|--------|---------|
| 1 | 工业废墟 | 橙+铁灰 | 传送带多、爆炸桶多 |
| 2 | 数据中心 | 蓝+白 | 能量屏障多 |
| 3 | 地下黑市 | 紫+金 | 商店概率翻倍 |
| 4 | 核心枢纽 | 红+黑 | 环境危害、Boss Rush |

---

## 交互地形
- 可破坏掩体：承受 N 发子弹后摧毁
- 能量屏障：定期开关，子弹可被反弹
- 传送带：推动实体沿方向移动
- 爆炸桶：被击中引爆，范围伤害
- 电网地板（第 4 层）：周期性通电

---

## 商店系统

### 碎片经济
- 普通敌人 1-3，精英 5-8，Boss 15-25 碎片
- 碎片飞行聚拢到玩家

### 商店界面
- 展示 4 个随机升级，刷新消耗 5 碎片（每次翻倍，离店重置）
- 购买后该槽位重新随机，价格 +20%

### 升级池
| 类别 | 升级 | 上限 |
|------|------|------|
| 武器 | 射速+15% | 5 层 |
| 武器 | 弹道+1 | 4 层 |
| 武器 | 弹速+25% | 3 层 |
| 武器 | 伤害+20% | 无上限 |
| 武器 | 弹道类型进化 | 不可回退 |
| 防御 | 护盾+1 | 3 层 |
| 防御 | 移速+10% | 5 层 |
| 防御 | HP+15% | 无上限 |
| 特殊 | 碎片+30% | 无上限 |
| 特殊 | 炮塔转速+30% | 4 层 |
| 特殊 | 无人机随从 | 2 个 |

## 局外成长

### 数据核心获取
- 每房间 +2，Boss +10，新深度层 +15（一次性），成就 +5

### 天赋树（三条线）
| 攻击线 | 防御线 | 通用线 |
|--------|--------|--------|
| 起始射速+10% | 起始HP+1 | 初始碎片+5 |
| 子弹伤害+5% | 护盾上限+1 | 刷新-1碎片 |
| 暴击 5% | 掩体减伤 15% | 起始弹道+1 |
| Boss 额外掉落 | 复活 1 次 | 解锁涂装 |

节点消耗递增核心解锁。

## 视觉风格

| 用途 | 色值 |
|------|------|
| 背景 | `#0a0a0f` |
| 玩家 | `#00ff88` |

### 渲染技术
- `globalCompositeOperation: 'lighter'` 霓虹叠加发光
- 实体 3 层绘制：外层光晕 → 中层半透明 → 内层实心
- 粒子系统：爆炸、碎片、尾迹
- 屏幕震动效果
- 坦克履带动画、能量屏障脉冲、门开关光效
